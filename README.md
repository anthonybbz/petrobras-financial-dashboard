# Petrobras — Análise Financeira 2024 vs 2025

Dashboard financeiro comparativo desenvolvido em Python e Excel,
analisando os principais indicadores financeiros da Petrobras.

## Indicadores analisados

- **Lucratividade:** Margem Bruta, Margem EBITDA, Margem Operacional, Margem Líquida
- **Retorno:** ROE, ROA, ROCE, ROIC
- **Liquidez:** Liquidez Corrente, Liquidez Imediata
- **Alavancagem:** Debt Ratio, Debt to Equity

## Ferramentas utilizadas

- Python (pandas, openpyxl)
- Excel (dashboard visual)
- Power Query

## Fontes dos dados

Demonstrações Financeiras oficiais extraídas do site da B3
- Empresa: Petrobras
- Período: Janeiro a Dezembro de 2024 e 2025

## Principal Análise

 A Margem Líquida aumentou de 7,5% em 2024 para 22,2% em 2025.
 A melhora foi impulsionada pela variação cambial, que em 2024
 gerou um prejuízo de R$ 60 bi, enquanto em 2025 reverteu
 para um ganho de R$ 20 bi.
